## What is Claw UI?

Claw UI is a modern web interface for OpenClaw that allows you to:

- **Chat with your AI agent** through an intuitive web interface
- **Connect multiple agents** from different devices
- **Access from anywhere** - no need to be on the same network
- **Secure cloud relay** - your conversations are routed through authenticated channels
- **Multi-session support** - manage multiple conversations simultaneously

## First Time Setup

### 1. Create Your Account

1. Navigate to the Claw UI URL (e.g., `https://claw-ui.yourdomain.com`)
2. Click **"Sign Up"** or **"Register"**
3. Enter your email address and create a password
4. Verify your email (if email verification is enabled)
5. Log in with your credentials

### 2. Initial Configuration

After logging in for the first time, you'll see the welcome screen. You have two options:

- **Run Setup Wizard** (recommended) - guided setup process
- **Manual Setup** - configure agents yourself

We recommend using the Setup Wizard for your first agent.

## Connecting Your First Agent

### Using the Setup Wizard

1. Click **"Run Setup Wizard"** from the home screen
2. The wizard will guide you through:
   - **Step 1: Name Your Agent** - Give your agent a memorable name (e.g., "Home Desktop", "Work Laptop")
   - **Step 2: Generate Token** - Click to create a secure authentication token
   - **Step 3: Install on Device** - Follow the instructions to install the OpenClaw plugin
   - **Step 4: Verify Connection** - Confirm your agent is connected

3. Copy the installation command and run it on your OpenClaw instance:

```bash
# Example installation command (generated by wizard)
openclaw plugin install claw-ui-cloud
openclaw config set claw-ui.relay-url "wss://relay.yourdomain.com"
openclaw config set claw-ui.auth-token "your-generated-token"
openclaw plugin enable claw-ui-cloud
```

4. Once connected, you'll see a green "online" status indicator

## Basic Usage

### Starting a Conversation

1. **From Home Page**: Click **"New Chat"** quick action
2. **From Sidebar**: Click the **"New Session"** button
3. **Keyboard Shortcut**: `Cmd/Ctrl + N` (if enabled)

### Sending Messages

- Type your message in the compose bar at the bottom
- Press `Enter` to send (or `Shift + Enter` for new line)
- Your agent will process and respond

### Managing Sessions

- **Switch sessions**: Click on session in sidebar
- **Rename session**: Click the session title at the top
- **Delete session**: Hover over session in sidebar â†’ click trash icon
- **Pin session**: Right-click session â†’ Pin (stays at top of sidebar)

### Agent Selection

If you have multiple agents connected:

- **Default agent**: Set in Settings â†’ Agent Preferences
- **Switch mid-conversation**: Click agent indicator in chat header
- **Agent-specific sessions**: Choose agent when creating new session

## Tips & Tricks

### Keyboard Shortcuts

- `Cmd/Ctrl + K` - Quick actions menu (if enabled)
- `Cmd/Ctrl + N` - New session
- `Cmd/Ctrl + ,` - Settings
- `Esc` - Close modals/dialogs

### Interface Customization

Visit **Settings** to customize:

- **Theme**: Light, Dark, or Auto (follows system)
- **Layout Density**: Comfortable (default) or Compact
- **Sidebar Width**: Adjust to your preference
- **Adaptive UI**: Let interface learn from your usage patterns

### Search & Filters

- **Search sessions**: Use search bar in sidebar
- **Filter agents**: Search in Agent Management page
- **Recent vs. Pinned**: Organize sessions for quick access

### Notifications

Enable notifications in Settings to get alerts for:

- Agent status changes (connect/disconnect)
- New messages in background sessions
- System updates

### Connection Troubleshooting

If your agent shows as offline:

1. **Check OpenClaw is running** on the device
2. **Verify plugin is enabled**: `openclaw plugin list`
3. **Check token**: Ensure auth token matches
4. **Test connection**: Use "Test Connection" button in Agent Management
5. **Check relay server**: Verify relay URL is correct
6. **Review logs**: Check OpenClaw logs for connection errors

### Session History

- View all past sessions in **History** page
- Search by date, content, or agent
- Export conversations (if supported)

### Security Best Practices

- **Use strong passwords** for your Claw UI account
- **Revoke unused agents** regularly
- **Don't share tokens** - each device should have its own
- **Enable 2FA** if available
- **Review connection history** periodically
